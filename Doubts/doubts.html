HTML DOM Element offsetTop

• The offsetTop property returns the top position (in pixels) relative to the parent.

• The returned value includes :
-- the top position, and margin of the element.
-- the top padding, scrollbar and border of the parent.

• The offset property is read-only.

Window pageXOffset

• The pageXOffset property returns the pixels a document has scrolled from the upper left corner of the window.

• The pageXoffset property is equal to the scrollX property.

• The pageXOffset property is read-only.
